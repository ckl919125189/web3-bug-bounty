## 如何开始

### 第一步：注册账号

```bash
# 推荐顺序
1. 注册 Immunefi（最全最多）
2. 注册 HackenProof
3. 注册 Bugcrowd
```

### 第二步：完善资料

- GitHub 个人主页
- 领英简历
- 安全研究背景

### 第三步：选择目标

```
优先级排序：
1. 你熟悉的 DeFi 协议
2. 高赏金项目
3. 低竞争项目
```

### 第四步：开始审计

```
审计流程：
1. 阅读项目文档
2. 分析智能合约代码
3. 查找漏洞
4. 编写漏洞报告
5. 提交
```

## 注意事项

### ⚠️ 法律风险

- ✅ 只测试有授权的项目
- ✅ 遵循平台规则
- ✅ 不要做未经授权的渗透测试

### ⚠️ 提交规范

- 详细描述漏洞
- 提供复现步骤
- 评估影响范围
- 给出修复建议

### ⚠️ 时间投入

- 首个漏洞可能需要数周-数月
- 需要持续学习
